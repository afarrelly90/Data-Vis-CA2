Portfolio Technical Documentation

This document outlines the usage of Bootstrap 5 elements and custom JavaScript implementations within the main index.html portfolio page.

The website is hosted at: https://afarrelly90.github.io/Data-Vis-CA2/

Bootstrap Components Used

Navigation Bar (Navbar)

The primary navigation bar uses the standard Bootstrap `navbar` component with the following customizations:

  • Sticky-top: Ensures the navbar remains visible when scrolling.
  • navbar-dark & bg-dark: Provides a dark theme against the light page content.
  • ms-auto: Uses utility classes to push the navigation links to the right.
  • Dynamic Class: The custom JavaScript Scroll-Spy applies the `active` class to `nav-link` elements.

Code Snippet:

<nav class="navbar navbar-expand-lg navbar-dark bg-dark sticky-top">
                <div class="container">
                    <a class="navbar-brand" href="#home">AF</a>
                    <div class="collapse navbar-collapse" id="navbarNav">
                        <ul class="navbar-nav ms-auto">
                            <li class="nav-item"><a class="nav-link" href="#about">About</a></li>
                        </ul>
                    </div>
                </div>
            </nav>

Contact Modal

A Bootstrap Modal component is used to display contact information, accessed by the buttons in the Hero and Contact sections.

  • Data Attributes: Triggered using `data-bs-toggle="modal"` and `data-bs-target="#contactModal"`.
  • Content: Includes an unordered list (`list-unstyled`) for clear contact details.

Code Snippet (Trigger Button):

<button type="button" class="btn btn-primary btn-lg" 
                data-bs-toggle="modal" data-bs-target="#contactModal">
                Contact Me
            </button>

Code Snippet (Modal Structure):

<div class="modal fade" id="contactModal" tabindex="-1" aria-labelledby="contactModalLabel">
                <!-- modal content here -->
            </div>

Cards (Skills, Experience & Projects)

Cards are heavily used for consistent, responsive grouping of content across the portfolio.

  • Card Header/Body/Footer: Used to separate the project details and grades.
  • h-100 & shadow-lg: Ensures all of the cards within a row have the same height and gives them a shadow.
  • Custom Class: The `hover-card` class applies a slight visual lift effect on hover (CSS customization).
  • Badges: Used extensively to categorize languages and technologies. In the Skills section, they use the `rounded-pill` style and distinct background colors (e.g., `text-bg-dark`, `text-bg-primary`) for visual grouping.

Code Snippet:

<div class="progress mb-4" role="progressbar" style="height: 6px;">
            <div class="progress-bar bg-success" style="width: 95%"></div>
            </div>

Layout, Grid, & Utilities

Grid System

The standard Bootstrap 12-column grid is used for all major layout areas, ensuring a fully responsive design across all devices.

  • Responsive Layouts: The Projects section uses `col-lg-4` (3-wide on large screens) and `col-md-6` (2-wide on medium screens).
  • Column Restriction: The About Me and Experience sections restrict content width using `col-lg-8` centered via `justify-content-center` to improve text readability.

Spacing and Utilities

Spacing utility classes were used a lot to maintain consistent vertical rhythm and component separation:

  • Vertical spacing: `py-5` is then applied to all of the main `<section>` tags to provide robust top/bottom padding for accurate Scroll-Spy detection.
  • Flexbox: `d-flex` and related classes (`gap-3`, `justify-content-between`, `align-items-start`) are used for internal alignment (e.g., placing the logo and text side-by-side in the Experience cards).

Linked Cards (Experience)

The entire Experience card is wrapped in an anchor tag to make the company logo and details fully clickable, linking out to the company's LinkedIn or website.

  • Structure: The `<div class="card">` is wrapped inside an `<a>` tag with `class="text-decoration-none d-block project-link"` to reset link styling and ensure the link spans the full card width.
  • Image Component: Company logos are included using the `<img>` tag with fixed height/width styles to maintain layout consistency.

Custom JavaScript Features

Time of day greeting

A function runs on page load to check the current hour and dynamically update the greeting message in the Hero section.

Code Snippet:

const hour = new Date().getHours();
                const hour = new Date().getHours();
                const greeting = document.getElementById('greeting');
                let greetingText = 'Hello,';

                if (hour < 12) {
                    greetingText = 'Good Morning,';
                } else if (hour < 18) {
                    greetingText = 'Good Afternoon,';
                } else {
                    greetingText = 'Good Evening,';
                }
                
                if (greeting) {
                    greeting.innerText = greetingText;
                }

Custom Scroll-Spy Implementation

A manual Scroll-Spy script is implemented to highlight the current section in the fixed navigation bar. This uses an event listener and includes fixes for the smaller sections such as the contact section (uses end of page detection).

Initially, the bootstrap scroll spy was considered to be used for this but it contained inaccuracies with the smaller sections

  • Offset: A fixed `+ 120` offset is added to `window.scrollY` to account for the sticky navbar height.
  • End-of-Page Fix: Special logic is used to force the 'Contact' link to be active when the user scrolls to the bottom of the page (within 5px of the maximum scroll).

Code Snippet:

const updateScroll =() => {
        const scrollPos = window.scrollY + 120;
        let currentSectionId = "home"; 
        sections.forEach((sec) => {
            const top = sec.offsetTop - 140; 
            const bottom = top + sec.offsetHeight;

            if (scrollPos >= top && scrollPos < bottom) {
                currentSectionId = sec.getAttribute("id");
            }
        });

        const pageHeight = document.body.offsetHeight;
        const viewportHeight = window.innerHeight;
        
        if (window.scrollY + viewportHeight >= pageHeight - 5) {
            currentSectionId = 'contact';
        }

        if (window.scrollY < 10) { 
            currentSectionId = 'home';
        }
        navLinks.forEach(link => { link.classList.remove("active"); });
        brandLink.classList.remove("active"); 

        if (currentSectionId === 'home') {
            brandLink.classList.add('active'); 
        } else {
            navLinks.forEach((link) => {
                if (link.getAttribute("href") === `#${currentSectionId}`) {
                    link.classList.add("active");
                }
            });
        }
    };
